# Black Ops 2 (T6) Zombies Scripts

These scripts are made for Black Ops 2 using Plutonium T6.  
Place them in your `Plutonium/storage/t6/scripts/zm/` scripts folder and they will load automatically when the map starts.

---

`zm_info.gsc`  
Adds chat commands like `.info`, `.help`, `.list`, and `.commands` to show AFK, bank, and server command lists.  
Also includes `.discord` and `.site` for community links.

`zm_lod_enforcer.gsc`  
Forces high model detail by adjusting LOD settings for sharper visuals.  
Improves appearance without affecting gameplay.

`zm_no_perk_limit.gsc`  
Removes the 4-perk limit and raises the cap to 99 so all perks can be collected.

`zombiescounter.gsc`  
Displays the number of zombies remaining during each round.  
Helps players and testers track progress and round pacing.
