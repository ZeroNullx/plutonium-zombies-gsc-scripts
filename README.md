# Call of Duty Zombies GSC Scripts

A simple collection of GSC scripts for Black Ops 1 (T5) and Black Ops 2 (T6).  
Each script is modular, lightweight, and made for custom Zombies servers or single-player testing.

---

## Supported Games
- **Black Ops 1 (T5)**
- **Black Ops 2 (T6)**

---

## Purpose
These scripts are designed for:
- Adding new chat commands and player tools  
- Unlocking gameplay limits (perks, rewards, etc.)  
- Testing or balancing maps  
- Enhancing visuals and HUD information  

---

## Directories
- **/bo1_scripts** – scripts built for Black Ops 1  
  - zm_aim_melee_bonus_points.gsc  
  - zm_health_zcount.gsc  
  - zm_no_perk_limit.gsc  
  - zm_secret_points.gsc  

- **/bo2_scripts** – scripts built for Black Ops 2  
  - zm_info.gsc  
  - zm_lod_enforcer.gsc  
  - zm_no_perk_limit.gsc  
  - zombiescounter.gsc
